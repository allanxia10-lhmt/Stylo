# Stylo

Stylo is an AI fashion and smart closet web app.

## Included

- Premium responsive web app UI.
- Email, Google, Apple, forgot-password demo flows.
- First-sign-in preferences and editable profile tab.
- Combined Closet + Outfits workspace.
- Visual search for clothing pieces inside outfit photos.
- Community outfit posts with 1-5 star ratings and comments.
- Try-on, shopping discovery, insights, and GitHub Pages deployment workflow.

## Build

```bash
npm run build
```

The build output is created in `dist/`.

## Deploy With GitHub Pages

1. Open repository settings.
2. Go to **Pages**.
3. Set **Build and deployment** source to **GitHub Actions**.
4. Run the **Build and Deploy Stylo** workflow or push to `main` after merging this branch.

## Branch

This app was uploaded on `stylo-app-build`. Merge it into `main` when ready.
